## 📝 Summary
비만의 위험성이 증가하면서 세계 각국은 비만을 예방하기 위해 노력 중이다. 런잇 앱을 통해서 오늘 운동한 칼로리 만큼의 음식을 
내가 아닌 런잇 캐릭터에게 먹여 성장 시킬 수 있다. 귀여운 음식 이미지와 캐릭터를 모으고 자랑하기 위해 재밌게 런닝을 할 수 있도록 도움을 줄 수 있다.
